# CLI

## 顶级命令

- `install`
- `shell`
- `extract`
- `mcp`

## `extract` 子命令

- `get <url> <output_file>`
- `post <url> <output_file>`
- `put <url> <output_file>`
- `delete <url> <output_file>`
- `fetch <url> <output_file>`
- `stealthy-fetch <url> <output_file>`

## 输出规则

- `.md`：Markdown
- `.html`：HTML
- `.txt`：纯文本
